# 🐾 반려동물 관리 시스템 (Pet Management System)

콘솔 환경에서 동물을 등록하고 상태를 관리할 수 있는 객체지향 프로그래밍(OOP) 기반 프로젝트입니다.

단순한 기능 구현을 넘어, 다양한 동물 종의 확장과 인터페이스를 통한 능력 분리 등 유연한 설계를 연습하기 위해 제작되었습니다.

---

## 📖 프로젝트 소개

이 프로젝트는 동물의 공통적인 속성을 추상화하고, 종별 특이 사항을 상속과 인터페이스로 해결해 나가는 과정을 담고 있습니다.

---

## ✨ 주요 기능

- **동물 관리**
    - 동물 등록
    - 동물 목록 조회
    - 상태(배고픔, 즐거움 등) 확인

- **상호작용**
    - 먹이 주기 (`Food` 시스템)
    - 놀아주기
    - 울음소리 출력

- **능력 확장**
    - `Flyable`, `Swimmable` 등 인터페이스를 활용한 특수 능력 실행

- **분류 시스템**
    - `Bird`, `Mammal`, `Reptile` 등 중간 계층을 통한 체계적 확장

---

## 🏗 설계 포인트

> **상속은 공통점을 묶고, 인터페이스는 능력을 부여한다.**

- **추상화 (`Animal`)**
    - 모든 동물이 가지는 이름, 나이, 상태값 등을 공통 멤버로 정의합니다.

- **계층형 구조**
    - 중간 분류(포유류, 조류 등)를 도입해 코드 재사용성과 확장성을 높였습니다.

- **인터페이스 기반 설계**
    - `날기`, `수영` 같은 특정 행동은 상속이 아닌 인터페이스로 분리하여 유연하게 조합할 수 있도록 했습니다.

- **객체 간 협력**
    - `Zoo` 클래스가 전체 흐름을 제어하며 객체 간 결합도를 낮추고 역할을 분리합니다.

## 🚀 시작하기

### 1. 환경 요구 사항

- Java JDK 11 이상
- Terminal 또는 Command Prompt

### 2. 설치 및 실행

```bash
# 1. 프로젝트 클론
git clone <repository-url>
cd <repository-folder>

# 2. 소스 코드 컴파일
javac src/*.java

# 3. 프로그램 실행
java src.Main## 🕹 실행 흐름

### 1. 동물 등록
이름과 종을 선택하여 새로운 동물을 등록합니다.

### 2. 목록 확인
현재 등록된 모든 동물의 상태를 확인합니다.

### 3. 상태 관리
먹이를 주거나 놀아주며 동물의 상태를 변화시킵니다.

### 4. 특수 능력 실행
날 수 있거나 수영할 수 있는 동물의 특별한 행동을 실행합니다.

## 📈 개발 로드맵 (Sprints)

| 단계 | 구분 | 주요 내용 | 상태 |
|------|------|-----------|------|
| Sprint 1 | 기초 다지기 | 기본 등록, 목록 확인, 먹이/놀이 시스템 | ✅ 완료 |
| Sprint 2 | 구조 고도화 | 종별 분류 확장, 인터페이스 도입, 먹이 적합성 판정 | ✅ 완료 |
| Sprint 3 | 시스템 확장 | 사육사(`Keeper`) 클래스, 전체 통계 기능, 예외 처리 강화 | 📅 예정 |
