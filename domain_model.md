# 도메인 모델 요약
---
## Animal
모든 동물의 공통 행동을 담는다.

### 속성
- `name` : 동물 이름
- `age` : 동물 나이

### 행위
- 자신의 종류를 반환한다: `getType()`

### 규칙
- 이름은 비어 있으면 안 된다.
- 나이는 0 이상이어야 한다.

### 검증
- `name == null` 이면 안 됨
- `name.isBlank()` 이면 안 됨
- `age < 0` 이면 안 됨

## Dog, Cat
`Animal`의 구현체

### 속성
- 부모인 `Animal`의 속성을 그대로 사용

### 행위
- `getType()`을 각자 다르게 구현
    - `Dog` → "강아지"
    - `Cat` → "고양이"

### 규칙


### 검증
- 생성 시 `Animal`의 공통 검증을 그대로 따름

## Zoo
동물들을 관리하는 관리자 역할.

### 속성
- `animals : ArrayList<Animal>`

### 행위
- 동물을 등록한다
- 등록된 동물 목록을 보여준다
- 목록이 비었는지 확인한다: `isEmpty()`

### 규칙
- 등록된 동물은 목록에 저장되어야 한다.
- 목록 출력 시 **번호가 1부터** 시작해야 한다.
- 목록 출력 시 각 항목은 **이름/종류/나이**가 표시되어야 한다.
- 목록이 비어 있으면 안내 문구가 출력되어야 한다.

### 검증
- `null` 동물은 등록하면 안 됨
